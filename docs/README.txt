CLEARWAVE AGENTE INSTALADOR - VERSÃO MVP

1. O que faz:
- Detecta novos relatórios (.xlsx ou .csv) em uma pasta definida
- Envia os relatórios automaticamente via e-mail ou API
- Roda diariamente como tarefa agendada no Windows

2. Como usar:
- Execute o instalador (interface.py)
- Selecione a pasta onde os relatórios são gerados
- A tarefa será criada automaticamente via agendador do Windows

3. Requisitos:
- Python 3.10+
- Pacotes: watchdog, tkinter
- Conexão com internet ativa
